# 🌊 Wavelength

> *Can you tune into the same frequency as your teammates?*

Wavelength is a **digital adaptation** of the acclaimed party game where players communicate through spectrums, intuition, and the art of finding just the right word. One player knows where the target lies — the rest must read their mind.

---

## 🎮 What is Wavelength?

Wavelength is a social deduction and communication game for **2–10 players**. Each round, one player becomes the **Psychic**: they see a hidden target on a semicircular dial positioned between two opposing concepts — like *Hot ↔ Cold* or *Underrated ↔ Overrated* — and must give a single clue that hints at its exact location.

The challenge? The clue must be a word or concept that lives *on that spectrum*, precise enough to guide the team to the right spot, yet abstract enough to spark debate.

---

## 🕹️ How to Play

### Setup

Each player receives one or more **180° dials**. Every dial shows:

- A **semicircle** with scored zones marked along the arc
- Two **opposing adjectives** at each end of the spectrum (e.g. *Soft* on the left, *Hard* on the right)
- A hidden **target zone** somewhere along the arc

### Phase 1 — The Psychic's Clue

The active player (the Psychic) sees where the target is on their dial. Their task:

1. Study the two opposing adjectives at each end
2. Think of a **single word or concept** that sits at that precise point on the spectrum
3. Announce the clue to the table — no gestures, no additional hints

> **Example:** Spectrum is *Quiet ↔ Loud*. The target is near the loud end.  
> The Psychic says: *"Rock concert"* 🎸

### Phase 2 — The Guess

The other players cannot see the hidden target. Based solely on the clue and the spectrum:

1. **Discuss** among yourselves where you think the target is
2. **Place your guess** on the dial — the closer to the center of the target zone, the more points you score
3. No numbers or percentages — only qualitative reasoning (*"more to the right"*, *"slightly left of center"*)

### Scoring

| Position | Points |
|----------|--------|
| 🎯 Center zone | 4 pts |
| Adjacent zone | 3 pts |
| Outer zone | 2 pts |
| Miss | 0 pts |

The opposing team may also earn **1 bonus point** by correctly guessing which *side* of the guess the true target lies on — so stay sharp even when it's not your turn.

### Winning

The first team to reach **10 points** wins. In case of a tie, teams play sudden death rounds until one pulls ahead.

## ✨ Features

- 🖥️ **Fully digital** — no physical components needed
- 🌐 **Multiplayer** — play with friends online or locally
- 🃏 **Large card pool** — dozens of spectrum pairs to keep every game fresh
- 🤝 **Cooperative mode** — all players work together against a shared goal
- 📱 **Responsive design** — works on desktop and mobile

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/tu-usuario/wavelength.git

# Navigate to the project folder
cd wavelength

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open your browser at `http://localhost:3000` and start playing.
